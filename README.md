#### Описание программы

Простое приложение по прогнозу погоды.

##### Использовал:
 1. React + redux-toolkit
 2. Typescript
 3. API openweathemap
 4. Formik
 5. Axios
 6. Scss

#### Клонировать этот репозиторий

````
$ git clone git@github.com:Evgeniy3/weather-app.git
````
#### Установить зависимости 

````
$ npm ci
````
#### Запустить приложение

````
$ npm start
````
